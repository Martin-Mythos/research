# Talking Breads V1 subject project

Minimal static Vite + React + TypeScript website used as the development subject for the Kun Chen agentic engineering tools study.

## Commands

```sh
npm install
npm run build
npm test
npm run smoke
npm run dev
```

Content is centralized in `src/content.ts` so a developer can update menu, hours, promotions, and contact data without changing component logic.
